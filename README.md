# Boilerplate with Babel Webpack Sass Openlayers 3

All you need to do is:
* `git clone https://github.com/stofolus/babel-webpack-sass-ol3.git`
* `cd babel-webpack-sass-ol3`
* `npm install`
* `npm start`

The following scripts are defined:
* `npm start` - Builds and runs in a local server at `http://localhost:8080`
* `npm run build` - Builds the application

If you want to create your own repository for your application you need to create your own git tracking
* `rm -rf .git`
* `git init`
